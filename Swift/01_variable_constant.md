## 1. 선언 방법
```swift
// 상수
let 이름: 타입 = 값
// 변수
var 이름: 타입 = 값
```

단, 값의 타입이 명확하다면 타입은 생략 가능
```swift
let 이름 = 값
var 이름 = 값
```

<br>

## 2. 초기화
차후 값을 할당하려는 경우 상수나 변수는 타입을 지정해주어야 함
```swift
let year: Int
var name: String

// 상수는 단 한번의 값 할당만 허용
age = 2023
name = "SSangRRae"
```
